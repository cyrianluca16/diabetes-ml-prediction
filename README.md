# Diabetes Prediction using Machine Learning

This project explores the use of machine learning algorithms to predict the risk of diabetes from medical data.

## Project Overview

The goal of this project is to build and evaluate classification models capable of predicting whether a patient is likely to develop diabetes based on several health indicators.

The project includes:
- Data preprocessing and cleaning
- Exploratory data analysis
- Training and comparison of multiple machine learning models
- Model evaluation and hyperparameter optimization

## Dataset

The dataset contains medical diagnostic measurements such as:

- Glucose level
- Blood pressure
- BMI
- Age
- Insulin level
- Pregnancies

The target variable indicates whether the patient has diabetes.

## Machine Learning Models

The following models were implemented and compared:

- Logistic Regression
- Support Vector Machine (SVM)
- Naive Bayes

Hyperparameters were optimized using GridSearch.

## Evaluation Metrics

Model performance was evaluated using:

- Accuracy
- Precision
- Recall
- F1-score
